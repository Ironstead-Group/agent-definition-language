# Standards Bodies — Reference

This folder holds one document per target standards organization (or per submission track). Each document should cover:

1. **Organization and project name** (e.g., Linux Foundation AAIF — Agentic AI Foundation, IETF, ISO/IEC JTC 1/SC 42).
2. **Relevant links** (charter, process, submission guidelines).
3. **Submission requirements** (format, IPR, copyright, language).
4. **Contacts / liaison** (if any; avoid listing private emails without permission).
5. **Status** (e.g., Not started, Pre-submission, Submitted, Published).
6. **Notes** (deadlines, dependencies on other bodies, parallel submissions).

## Current Files

- **[linux-foundation-aaif.md](./linux-foundation-aaif.md)** — Linux Foundation / AAIF (Agentic AI Foundation) track.
- **[ietf-rfc.md](./ietf-rfc.md)** — IETF RFC track.
- **[iso.md](./iso.md)** — ISO/IEC (e.g., JTC 1/SC 42) track.

Add or update files as new bodies are added to the roadmap.
